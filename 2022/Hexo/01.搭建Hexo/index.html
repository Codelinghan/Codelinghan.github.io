# 在 github 上创建并设置远程库

1. 选择 New repository，
2. 在 Repository name 下面填写你要创建的地址，这个地址是就是你的域名，以 github.io 结尾。
例如：填写 "wapchief.github.io"以后，在地址栏直接输入"wapchief.github.io" 就可以访问网站，
按照提示一步步完成创建。
3. 创建之后，点击 setting 设置远程库：

(1) 找到 "GitHub Pages" 点击 "Automatic page generator" 等待 github pages 创建完成。
(2) 输入你创建的地址测试 github pager 是否创建成功。此时页面是空的没有任何东西。

作者本人的操作是：

  • 1. 选择 New repository

  • 3. 选择 public
  • 4. 点击新建仓库

# 部署 Blog 到互联网

【1】 打开 Blog 文件夹，打开 “_config.yml” 文件
修改最后的 deploy 为

  deploy:
       type: git
       repo: https://github.com/Codelinghan/Codelinghan.github.io.git
       branch: main

   注意：
         repo: https://github.com/Codelinghan/Codelinghan.github.io.git
              （https://github.com/Codelinghan/Codelinghan.github.io.git 这个链接是Github的仓库地址）
